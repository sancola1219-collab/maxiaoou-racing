# 馬小歐賽車 🏎️

3D 卡丁賽車網頁遊戲 — 16 條賽道 × 4 大盃賽 × 8 位車手。

**▶ 立即遊玩：https://sancola1219-collab.github.io/maxiaoou-racing/**

## 特色

- 🏆 **盃賽模式**：蘑菇盃、花之盃、星星盃、閃電盃，各 4 場積分賽
- 🏁 **單場競賽**：16 條主題賽道任選（草原、沙漠、雪山、火山、天空之城、彩虹之路…）
- ⏱️ **計時挑戰**：單人挑戰最佳紀錄
- 🎁 **7 種道具**：蘑菇、綠/紅龜殼、香蕉、星星、閃電、金幣（依名次加權抽選）
- 💨 **甩尾噴射**：蓄力火花藍→橘，放開爆衝
- 🪙 **金幣系統**：越多極速越快，被撞會噴掉
- 🤖 7 台 AI 對手（橡皮筋難度）、📱 支援觸控、🔊 全合成音效

## 操作

| 動作 | 按鍵 |
|---|---|
| 加速 / 煞車 | ↑↓ 或 W S |
| 轉向 | ←→ 或 A D |
| 跳躍甩尾 | Shift / 空白鍵 |
| 使用道具 | Ctrl / E / Enter |
| 暫停 | Esc |
| 靜音 | M |

## 技術

純前端 vanilla JS + Three.js（r149，已打包在 repo，可離線玩）。
賽道、車輛、裝飾全部程序化生成，零圖片素材。
開發交接見 `HANDOFF.md`。

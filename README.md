# 馬小歐賽車 🏎️

3D 卡丁賽車網頁遊戲 — 24 條賽道 × 6 大盃賽 × 8 位車手 × 14 種道具 × 動態天氣。

**▶ 立即遊玩：https://sancola1219-collab.github.io/maxiaoou-racing/**

## 特色

- 🏆 **盃賽模式**：蘑菇盃、花之盃、星星盃、閃電盃、月亮盃、王冠盃，各 4 場積分賽
- 🏁 **單場競賽**：24 條主題賽道任選（草原、沙漠、雪山、火山、天空之城、櫻花公園、水底世界、幽靈鬼宅、星際基地、彩虹之路…）
- ⏱️ **計時挑戰**：單人挑戰最佳紀錄
- 🌦️ **動態天氣**：陣雨、雷暴閃電、飄雪、暴風雪、濃霧、沙塵暴、火山灰、落櫻、洋流氣泡 — 每場隨機，影響抓地力與能見度
- 🎁 **14 種道具**：蘑菇/三重蘑菇/黃金蘑菇、綠/紅/藍龜殼、香蕉、假道具箱、炸彈、墨魚、火箭衝刺、星星、閃電、金幣（依名次加權抽選）
- ✨ **華麗特效**：噴射火焰、甩尾煙、星星彩虹拖尾、爆炸爆裂、速度線、閃電閃屏
- 💨 **甩尾噴射**：蓄力火花藍→橘，放開爆衝
- 🪙 **金幣系統**：越多極速越快，被撞會噴掉
- 🤖 7 台 AI 對手（橡皮筋難度）、📱 支援觸控、🔊 全合成音效

## 操作

| 動作 | 按鍵 |
|---|---|
| 加速 / 煞車 | ↑↓ 或 W S |
| 轉向 | ←→ 或 A D |
| 跳躍甩尾 | Shift / 空白鍵 |
| 使用道具 | Ctrl / E / Enter |
| 暫停 | Esc |
| 靜音 | M |

## 技術

純前端 vanilla JS + Three.js（r149，已打包在 repo，可離線玩）。
賽道、車輛、裝飾全部程序化生成，零圖片素材。
開發交接見 `HANDOFF.md`。

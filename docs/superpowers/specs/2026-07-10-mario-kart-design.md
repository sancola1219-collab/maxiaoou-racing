# 馬小歐賽車 — 設計文件（2026-07-10）

## 目標
做一款類似 3D 瑪利歐賽車的網頁遊戲，關卡越多越大越好。純前端、自包含（Three.js 打包進 repo），可離線玩、可上 GitHub Pages。

## 使用者未回覆時採用的預設
- 16 條賽道 / 4 個盃賽（蘑菇盃、花之盃、星星盃、閃電盃）
- 完整玩法：道具系統、7 台 AI 對手、甩尾噴射、金幣
- Three.js r149 UMD 打包進專案（js/three.min.js）
- 完成後照慣例部署 GitHub Pages

## 遊戲模式
1. **盃賽（GP）**：4 場積分賽（15/12/10/8/7/6/5/4 分），最後總排名頒獎
2. **單場競賽**：任選 16 條賽道之一
3. **計時挑戰**：單人 + 3 顆蘑菇，最佳紀錄存 localStorage

## 玩法系統
- **駕駛**：加速/煞車/轉向；跳躍甩尾蓄力 → 藍火/橘火噴射
- **道具**（依名次加權抽選）：蘑菇、綠龜殼、紅龜殼（追蹤）、香蕉、星星、閃電、金幣
- **金幣**：撿金幣提升極速（上限 10 枚），被撞掉 3 枚
- **地形**：出賽道變慢；天空/彩虹賽道會掉落虛空 → 重生；冰面抓地力低
- **8 位角色**：速度/加速/操控/重量各有差異

## 技術架構（一檔一責）
| 檔案 | 職責 |
|---|---|
| `js/data/tracks.js` | **資料層**：16 賽道控制點、16 主題、4 盃賽、8 角色。擴充改這裡 |
| `js/track.js` | Catmull-Rom 取樣、賽道網格/裝飾/小地圖生成、最近取樣點查詢 |
| `js/kart.js` | 賽車物理（速度/甩尾/碰撞/圈數）+ 賽車 3D 模型 |
| `js/ai.js` | AI：沿樣本點前瞻導航、彎道減速、橡皮筋、用道具 |
| `js/items.js` | 道具箱、金幣、投射物（龜殼/香蕉）、雷擊/星星效果 |
| `js/audio.js` | WebAudio 合成音效 + 引擎聲 + 簡單 BGM，可靜音 |
| `js/ui.js` | DOM 選單（標題/模式/角色/盃賽/賽道/結算）+ HUD + 小地圖 + 觸控 |
| `js/game.js` | 主迴圈（固定步長）、比賽流程、攝影機、排名 |

- 賽道表示法：閉合 Catmull-Rom 曲線（控制點 [x,z,高度]）→ 密集取樣（位置/切線/左向量）→ 由此生成路面網格、地面高度、橫向偏移、進度/圈數、小地圖
- 測試掛鉤：`window.GameTest.step(dt)` 手動推幀 + 像素取樣（背景分頁測試法）
- 觸控裝置顯示虛擬按鈕（自動加速選項）
